## Requirements

* install [vscode](https://code.visualstudio.com/)
* install [android studio](https://developer.android.com/studio)

## Installation

Install react-native-course-delivery with npm or yarn

in case of npm:
```bash
  cd react-native-course-delivery
  npm install
  npm run android
```

in case of yarn:
```bash
  cd react-native-course-delivery
  yarn
  yarn android
  yarn start
```
